# Fitlog

## アプリケーション概要
Fitlogは、日々の筋トレや有酸素運動の記録を簡単に管理できるアプリケーションです。  
トレーニング内容や消費カロリーを記録し、振り返りやモチベーション維持に役立てることができます。

## 利用方法
1. ユーザー登録またはログインを行う  
2. 「新規作成」からワークアウトを登録  
3. 筋トレ・有酸素運動の内容を入力  
4. 一覧ページで履歴を確認  
5. 詳細ページで内容を確認・編集・削除
6. マイページでの他ユーザー含む消費カロリーランキングの閲覧  

---

## アプリケーションを作成した背景
筋トレを継続する中で、日々のトレーニング内容を記録し振り返ることでモチベーションを維持したいと考えました。  
しかし、既存のアプリでは操作が複雑だったり、必要な情報がすぐに確認できないと感じることがありました。  

そこで、シンプルで使いやすく、トレーニング記録に特化したアプリを作成しました。
また、他ユーザーとの消費カロリーランキングも記載することで、モチベーションの維持しやすいようにしました。

## 工夫したポイント
- 筋トレと有酸素を分けたデータ設計により、柔軟な記録が可能
- UIをシンプルにし、直感的に操作できるよう設計
- カロリーを自動計算し、ユーザーの負担を軽減
